Cards Against the World

HTML | CSS | Bootstrap | JavaScript | jQuery | jQuery UI | jQuery Flip | Font Awesome

Cards Against The World is a clone of the popular adult card game Cards Against Humanity. It has a reduced set of cards and is limited to two players and a PC player who contributes cards to simulate a four player game.

  • Date: Feb 2016
  • GitHub: Code




The Whisky Stone

HTML | CSS | Materialize | JavaScript | jQuery | Node | Express | Postgresql | JDOM | Flickr

The Whisky Stone is an application that allows a user to create a profile, search for single malt Scottish whisky and create a list of their favorites. They can tag the whisky with key words which are visible to other users. The Whisky data was scrapped from a Whisky database.

  • Date: Feb 2016
  • GitHub: Code




Rainy Day App

HTML | CSS | Bootstrap | Ruby | Rails | Postgresql | Sequelize | Google Places

Rainy Day is an application to help users find places they haven't been and save them to a profile, creating a list of places to visit when they have spare time. In addition they can add goals to accomplish at each location, like climb to the top of snake pass. Once they accomplish their goals they can unlock achievements which are listed on their profile.

  • Date: Mar 2016
  • GitHub: Code




The Wizards Unicorn

HTML | CSS | Bootstrap | JavaScript | Angular | Node | Express | MongoDB | Microsoft Voice to Text

The Wizards Unicorn is a text adventure game controlled by the users voice. The user can play by speaking in to a mic or by typing commands if the API can't recognize their speech. The text adventure was written using JavaScript and Angular. Microsoft's Voice to Text api was used to convert the users voice in to text.

  • Date: April 2016
  • GitHub: Code




Seattle Restaurant Week

HTML | CSS | Bootstrap | JavaScript | Angular | Node | Express | MongoDB | CHEERIO

Restaurant Week is a hub for the restaurants taking part in Seattle's Reasaurant Week 2016. The restaurant data is scrapped using Cheerio and stored in MongoDB. The user is able to sign in and see a list of all available restaurants. COMING SOON - Search capability.


  • Date: Mar 2016
  • GitHub: Code




Short.ify

HTML | CSS | Materialize | JavaScript | jQuery | Node | Express | PostgreSQL | sequelize

Short-ify is a simple link shortener. After short links have been created the number of times each one is used is tracked and a list of links ordered by popularity is created.


  • Date: Feb 2016
